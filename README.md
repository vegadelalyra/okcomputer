# okcomputer

Get quotes of any words you send to the terminal!

REMEMBER: THE COMMAND IS "okcomputer".
(hail radiohead)

## Install

```npm
npm install -g okcomputer
```

# Usage

```bash
 okcomputer <keywords>
```
or 

```bash
 okcomputer
 I do listen, human: <keywords>
```
